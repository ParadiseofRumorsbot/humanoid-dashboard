⚠ 풀(Pool) 중복합산 금지 — 반드시 먼저 읽기
아래 대당 ASP는 같은 풀 안에서 절대 단순 합산하지 말 것. 휴머노이드 BOM에서 액추에이터 풀은 전체의 약 56%를 차지하며, 이 풀 안에 이미 모터·감속기·볼/유성롤러 스크류·영구자석·관절 센서가 모두 포함됨. 따라서 모터+감속기+스크류+자석+센서 개별 추정치를 더하면 동일 콘텐츠를 여러 번 세는 이중계상이 발생함. 칩(컴퓨팅·전력반도체)·상류 소재(희토류·자석 원소재)는 별개 풀이나, 풀 간에도 일부 기능 중복(예: 전력반도체 vs 게이트드라이버)이 존재할 수 있음.
기준: 휴머노이드 1대 양산 정착 시 BOM ~US$25,000(약 3,400만원, 1,360원/$ 가정). 모든 수치는 BOM 역산 기반 자체추정([자체추정])이며 각 사 미공시임. 현 시점 휴머노이드향 실매출은 대부분 0(설계 채택 전 단계)이라는 점을 캐비엇으로 명기함. 통화 병기(USD). 판정: 적합 = 온로봇 BOM에 직접 들어갈 개연성 높음 / 부분 = 조건부·옵션·간접 / 의문 = 온로봇 BOM 근거 약함.
